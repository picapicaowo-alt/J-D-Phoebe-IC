# Genesis (DESIGN.md kit)

Source: [designmd.ai/chef/genesis](https://designmd.ai/chef/genesis) — downloaded for local reference. MIT license.

## Overview

An editorial precision interface for a community platform where developers discover, share, and download design system files. The aesthetic is quietly confident — bold display typography, generous spacing, and gallery-frame card surfaces.

## Colors

- Primary (#6366F1): CTAs, active states, links, focus rings — indigo
- Primary Hover (#4F46E5)
- Secondary (#20970B): brand highlight (use sparingly)
- Neutral (#9C9C9C): muted UI
- Background (#FAFAFA)
- Surface (#FFFFFF)
- Text Primary (#0A0A0A)
- Text Secondary (#6B6B6B)
- Border (#E8E8EC)
- Success (#10B981), Warning (#F59E0B), Error (#EF4444)

## Typography

- Display / headings: **General Sans** (Fontshare)
- Body / UI: **DM Sans** (Google Fonts)
- Code: **JetBrains Mono**

Type scale: Display 72px, Headline 60px, Section 32px, Subhead 24px, Body 15px, Small 13px, Caption 12px, Overline 11px uppercase.

## Components

- Buttons: primary indigo fill, white text, **6px** radius, medium weight. Secondary: transparent + 1px border. Ghost: text only. Destructive: red border + text.
- Cards: white surface, 1px border, **12px** radius; hover lift ~2px + subtle shadow.
- Inputs: 6px radius, 10px × 14px padding, 14px font; focus indigo border + 3px ring.
- Chips: pill, gray-100 / active indigo.

## Spacing

Base **4px**; scale 4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80, 96px. Container max **1280px**, horizontal padding **24px**.

## Do / Don’t (summary)

- Use indigo only for interactive elements.
- Keep 4px grid; General Sans for headings, DM Sans for body — do not swap.
- Do not use pure #000 / #fff for text; avoid decorative gradients.
